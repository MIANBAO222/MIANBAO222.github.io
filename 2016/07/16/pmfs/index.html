基本教程

1.先把pmfs从项目中git下来
Git clone pmfs

2.复制到usr/src(视喜好而定，视具体情况而定)

3.切换为root权限，或者全程使用sudo
4.找到pmfs并cd到此目录下，然后make menuconfig（配置内核），这一步需要依赖
在Ubuntu 下 以来可以由指令

来安装
centos下 以来可以由指令

来安装
然后进行内核配置

{make menuconfig中的设置}（pmfs必要，其他操作默认吧大概）
要善用搜索功能
pmfs[y]
pmfs—test[n]
pmfs——test——module[n]
pmfs——write——protect[y]
pmfs-xip[y]
5安装内核，首先gcc什么的肯定是必要的，依赖参考百度的内核编译，
可采用两种方式

或者
j后面的视内核数而定

5.更新grub(上一步已经自动更新了)
6.切换内核
8.报错–文件系统类型错误什么的
9.提示可以使用dmesg|syslog查看错误
10.发现提示要安装 nss-myhostname
nss-myhostname是为了支持dmesg的
11.此时更新了很多软件（apt-get）
12.然后安装

14.出现报错ioremap of the pmfs image failed（1）
16.解决问题：
在grub中预留内存
在linux那行里面加入memmap=2G$4G 留下2g运存其余备用不准用
memmap=2G$4G一定要加不然会进call trace
上面那个结论是错误的
7.执行mount
